# What Is Z-TSI
- Z-TSI Is An Termux Shortcut Keys Installer Coded By Zulf-135

# How To Use
- Just Copy Following Command And Run It In Termux One By One
- $ pkg install php 
- $ pkg install git
- $ cd $HOME
- $ git clone https://github.com/Zulf-135/Z-TSI
- $ php $HOME/Z-TSI/Configuration.php
